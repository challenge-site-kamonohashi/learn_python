"Terminalから"cpしたり、GUIでコピペするのは禁止です。

真面目に全部書いてもいいですし、
Vimのコマンドを調べて楽してもいいです。(ただし、ある程度タイピング速度を鍛えておいてください)


homework/1st/に自分のディレクトリ(英字で)を作り、
以下のファイルをgithubにpull requestしてください.
・homework.py   (gen_map.pyをそっくり書いてください.)
・homework.png  (gen_map.pyを実行した結果をスクショしてください.)
・method.txt    (どういうコマンドを使ったか書いてください.))


=====gen_map.pyを実行する方法=====
同じディレクトリに移動して、
$ python gen_map.py
-----中断する方法-----
実行しているTerminalで
Ctrl + c


=====スクショのやり方=====
-----全画面スクショ-----
Ctrl + PrtScnSysRq(っていうキーがある)
-----部分スクショ-----
Shift + PrtScnSysRq
カーソルが十字になるからマウスで指定
-----ウィンドウをスクショ-----
ウィンドウをクリックして、
Alt + PrtScnSysRq
